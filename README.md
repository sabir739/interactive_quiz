# 📘 Interactive Quiz Web Application

## 📌 Project Overview

This project is a simple and interactive quiz application developed using **HTML, CSS, and JavaScript**. It allows users to answer multiple-choice questions and view their final score along with feedback.

---

## 🎯 Objective

The objective of this project is to understand:

* Basic web development structure
* DOM manipulation using JavaScript
* Event handling and user interaction
* Styling using CSS for better user experience

---

## 🛠️ Technologies Used

* HTML5 (Structure)
* CSS3 (Styling & Layout)
* JavaScript (Logic & Interactivity)

---

## 📂 Project Structure

```
interactive_quiz/
│
├── index.html      # Main structure of the quiz
├── style.css       # Styling and layout
└── script.js       # Quiz logic and interactivity
```

---

## 🚀 Features

* Multiple-choice quiz questions
* Dynamic question loading
* Score tracking system
* Result display with feedback:

  * Excellent
  * Good Job
  * Try Again
* Restart quiz functionality

---

## ▶️ How to Run the Project

1. Download or clone the repository
2. Open the folder `interactive_quiz`
3. Double-click on `index.html`
4. The quiz will open in your browser
